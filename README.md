# RealFutharkLanguage

(Þat er einungis kallað "futhark" ef það er galdrar sem eru fluttir af víkingum, en ef eigi, þá er þat glitrandi fjölrætt, hátt stig, fúnksjónal, gögn samhliða, raðúthlutunartungumál).
